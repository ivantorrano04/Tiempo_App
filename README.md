# SmartWeather Pro
### The Ultimate Weather Experience

A professional-grade weather application built with Vanilla JS, designed for simplicity and aesthetic excellence.

## Features
- **Real-time Weather Data**: Aggregated from OpenWeatherMap, WeatherAPI, and Open-Meteo for maximum accuracy.
- **Dynamic Theming**: The interface adapts to the current weather conditions with beautiful gradients and animations.
- **Hourly Forecast**: Interactive charts powered by Chart.js.
- **5-Day Prediction**: Comprehensive outlook.
- **Deep Insights**: UV Index, Wind Direction, Humidity, Visibility, and more.
- **Glassmorphism UI**: A modern, sleek interface with blur effects and smooth transitions.

## Tech Stack
- **HTML5 & CSS3** (Variables, Flexbox, Grid)
- **JavaScript (ES6+)**
- **Chart.js** for visualization
- **Phosphor Icons** for iconography

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enjoy!
